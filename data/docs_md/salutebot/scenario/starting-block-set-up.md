По умолчанию сценарий чат-бота запускается с экрана, который расположен в верхнем левом углу редактора Graph. Начальный экран отмечается значком **Начало**.

Так, если в левом верхнем углу сценария расположен экран с блоком [**Текст**](blocks/text.md) и сообщением «Привет, Мир!», при получении [команды запуска](../dialog/start.md), чат-бот начнет разговор с пользователем с фразы «Привет, Мир!».

Вы можете изменить такое поведение и назначить начало сценария на любой экран сценария. Для этого:

1. Откройте сценарий в редакторе Graph.
2. Наведите курсор мыши на экран, который нужно сделать начальным.
3. Нажмите кнопку **···**.

   Откроется меню настроек экрана.
4. Выберите пункт **Сделать началом сценария**.

Начальный экран отметится значком **Начало**. При запуске чат-бота сценарий будет начинаться с заданного экрана.

Нельзя начинать сценарий с экрана, который содержит блок **Завершение сценария**.

## Настройка вызова начального экрана

Вы можете настроить вызов начального экрана при получении [незнакомой фразы](../dialog/unknown-phrases.md) на запуске чат-бота.

Если параметр включен, при получении неизвестной фразы на запуске чат-бота будет открываться начальный экран, а не срабатывать реплика из раздела **Ответы чат-бота при форс-мажорах**. Такое поведение актуально если пользователь запускает чат-бот впервые или по истечении заданного времени.

По умолчанию параметр выключен.



Чтобы настроить вызов начального экрана при получении неизвестной фразы:

1. Откройте сценарий в редакторе Graph.
2. Наведите курсор мыши на начальный экран.
3. Нажмите кнопку **···**.

   Откроется меню настроек экрана.
4. Выберите пункт **Настроить условия старта**.
5. Включите переключатель **Переводить на начало сценария, если не удалось распознать фразу**.
6. Задайте время, по истечении которого, при получении незнакомой фразы на запуске чат-бота будет открываться начальный экран.
