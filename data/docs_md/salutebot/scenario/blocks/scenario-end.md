# Завершаем сценарий

Обновлено 21 февраля 2024

[![](/assets/salutebot/scenario/blocks/scenario-end/salutebot-new.png)
SaluteBot](../../overview.md)[![](/assets/salutebot/scenario/blocks/scenario-end/Graph.png)
Graph](https://developers.sber.ru/docs/ru/va/graph/overview)

Блок **Завершение сценария** предназначен для завершения работы чат-бота.

При выполнении этого блока чат-бот удаляет все данные текущей сессии пользователя и завершает диалог с пользователем. При этом окно чат-бота в интерфейсе остается открытым.
