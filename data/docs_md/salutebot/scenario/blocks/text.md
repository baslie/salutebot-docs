Блок **Текст** предназначен для отображения ответов бота в чате с пользователем.

![Реплика чат-бота в интерфейсе Салют](/assets/salutebot/scenario/blocks/text/va_text_phrase-a3b1eb6e98c1ccf2500fb68144265b5f.png)

В поле **Реплика чат-бота** указывается текст, который отобразится в чате. Длина реплики чат-бота должна быть менее 1000 символов и может содержать [переменные](https://developers.sber.ru/docs/ru/va/graph/variables/create-variables).

С помощью кнопки **Добавить другую реплику** в блок можно добавить неограниченное количество реплик. Чат-бот выбирает реплики случайным образом.

На один экран сценария можно добавить несколько блоков **Текст**. Чат-бот показывает реплики из каждого блока в отдельном сообщении.

![Несколько блоков Текст в интерфейсе Салют](/assets/salutebot/scenario/blocks/text/several_text_blocks-169ac2f1161a5803e836b4f6dc9bc72e.png)

## Форматирование текста

Для удобства формирования ответа в разделе **Настройки** доступны два режима:

* Простой текст, когда ответ формируется без поддержки форматирования и разметки (по умолчанию);
* Режим разметки текста сообщения, включающий форматирование текста с помощью кнопок.

При использовании режима разметки следует выделить часть текста и применить к ней выбранный стиль форматирования. В результате текст в интерфейсе будет отображен согласно указанному форматированию и после сохранения показан в сценарии.

Доступны следующие элементы форматирования:

* Жирный шрифт;
* Курсив;
* Зачеркнутый шрифт;
* Ссылка;
* Маркированный список.

Используемая разметка может частично не поддерживаться в некоторых каналах, подключенных в Jivo.
