Для публикации проектов SaluteBot в каналах Jivo вам потребуется аккаунт в Jivo. Если аккаунта нет, [зарегистрируйтесь](https://app.jivosite.com/signup.html) . Для работы с проектом подходят любые версии аккаунта Jivo.

Для подключения проекта к аккаунту Jivo:

1. Откройте проект SaluteBot.
2. В разделе **Чат-платформа** выберите **Jivo** нажмите **Далее**.

   Откроется окно **Добавление интеграции Jivo**.
3. Укажите почту и пароль своего аккаунта Jivo и нажмите кнопку **Войти**.

## Проект в Личном кабинете Jivo

1. Авторизуйтесь в Личном кабинете Jivo.
2. Откройте **Управление** → **Расширения** → **Интеграции**.
3. Создайте JivoBot со сценарием **Готовые ответы**.

Проект появится в вашем пространстве в Studio.

Проект нельзя редактировать. Для внесения изменений нажмите кнопку **Копировать проект**. В скопированном проекте вы можете создать новый сценарий, подробнее об этом читайте в разделе [Создаем сценарий](bot-steps/scenario-creation/scenario-creation-guide.md). Для изменения текущего сценария, нажмите кнопку **Изменить сценарий**, и на его основе будет создан проект Graph, доступный для редактирования.

## Как подключить бота в канал

Каналы — это способ общения клиентов с ботами. Это могут быть соцсети, например ВКонтакте, мессенджеры (Telegram, WhatsApp), почта, сайт и другие ресурсы.

Полный список каналов для подключения SaluteBot вы найдете в личном кабинете Jivo в разделе **Управление** > **Каналы связи**.

## Как настроить каналы в Jivo

Для подключения бота в один из каналов, доступных в Jivo:

1. Откройте проект SaluteBot.
2. В разделе **Каналы для публикации** нажмите кнопку **Изменить** и перейдите в личный кабинет Jivo.
3. Выберите **Управление** → **Расширение** → **Подключенные**.
4. Нажмите кнопку **Опции** в строке с ботом.
5. Выберите в списке каналы, которые необходимо подключить к боту.

   Подробную информацию о подключении каналов ищите в [Базе знаний Jivo](https://www.jivo.ru/help/) .
6. Нажмите кнопку **Сохранить**.

После этого бот будет работать в подключенных каналах Jivo.

Один канал можно подключить только к одному SaluteBot.

Возможна так же интеграция с [платформой EDNA](edna-integration.md) и [омниканальным решением Webim](webim-integration.md).
