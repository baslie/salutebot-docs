С помощью автогенерации [сущностей](../bot-development/entities/overview.md) вы можете упростить добавление примеров синонимов.

Количество попыток автогенерации ограничено условиями тарифа.
На бесплатном тарифе доступно не более 5 попыток для всех проектов.

Чтобы сгенерировать примеры для сущности:

1. Откройте сценарий бота в Graph.
2. Перейдите в раздел **Сущности**.
3. Нажмите кнопку **Создать сущность**.
4. Укажите ее параметры: название и описание. Например, названием может быть "Auto", а описанием - "Марки немецких автомобилей".
5. Нажмите **Сгенерировать словарь**.
6. Дождитесь окончания процесса, в результате которого вам будут представлены сгенерированные с помощью нейросети записи и синонимы, подходящие под тематику вашей сущности. Например, для сущности "Auto" записями будут "БМВ", "Мерседес", "Опель", а синонимами - "bmw", "бэха" и т.д.

Обратите внимание: при повторной генерации сущность будет сгенерирована заново, а ранее добавленные записи будут сброшены. Количество попыток автогенерации при этом уменьшится.

7. Сохраните справочник.

Чтобы сгенерировать синонимы для одной записи:

1. Добавьте в сущность новую запись.
2. Укажите ее название, например, "БМВ" и нажмите **Enter**.
3. Выберите в меню генерации в поле создания синонимов **Сгенерировать синонимы**.
4. В результате будут добавлены синонимы только для выбранной записи, например "bmw", "бумер", "бэха".

Обязательно сохраните сгенерированные сущности перед выходом со страницы - автоматически результаты не сохраняются!

## Автоматическое заполнение сущностей с помощью GigaChat

Для более точной и корректной настройки сущности возможно добавлять переданные ранее синонимы и записи к существующей сущности или сгенерировать сущность заново.

При генерации сущности, для которой ранее были добавлены примеры и синонимы, новые сгенерированные записи дополнят существующие. Дублирующие записи и синонимы добавлены не будут.

Если сущность будет сгенерирована заново, добавленные ранее записи в ней будут очищены.

Для генерации сущности с учетом добавленных записей:

1. Откройте сценарий в Graph.
2. Перейдите в раздел **Сущности**.
3. Нажмите **Создать сущность**.
4. Если вам нужно заполнить сущность полностью и создать запись и синонимы к ней по указанному описанию, нажмите **Сгенерировать словарь**.
   Если требуется заполнить синонимами выбранную запись в существующей сущности, выберите запись в **Словаре синонимов** и нажмите **Сгенерировать синонимы**. В этом случае запись будет заполнена на основании первого значения записи в сущности.
5. Сохраните сценарий.

В результате успешного создания будут отображены как ранее добавленные записи так и список новых записей и синонимов сущности.

Количество попыток генерации ограничено условиями тарифа.

Для генерации сущности с учетом добавленных записей:

1. Откройте сценарий в Сode.
2. Перейдите в раздел **Сущности**.
3. Добавьте новую сущность или откройте уже существующую.
4. Заполните поле **Тематика**.
5. Добавьте несколько записей и синонимов к ним.
6. Нажимает **Генерировать записи**.
7. Сохраните сценарий.

В результате успешного создания будут отображены как ранее добавленные записи так и список новых записей и синонимов сущности.
