[Studio](https://developers.sber.ru/studio/login) — ваша точка входа в управление чат-ботом.

Зарегистрироваться в личном кабинете [Studio](https://developers.sber.ru/studio/) можно по Сбер ID, по СберБизнес ID или по почте.

## По Сбер ID

Чтобы зарегистрироваться в Studio по Сбер ID:

1. Откройте [Studio](https://developers.sber.ru/studio/) .
2. Нажмите кнопку **Войти по Сбер ID**.
3. Введите номер телефона российского оператора и нажмите кнопку **Войти или создать Сбер ID**.

Если ваш телефон привязан к приложению СберБанк Онлайн, вы получите push-уведомление. Нажмите на него, чтобы подтвердить вход в Studio. Если приложения СберБанк Онлайн нет, вам придет SMS-код. Введите его и нажмите кнопку **Подтвердить**.

На этом регистрация завершается. Теперь вы можете авторизоваться в Studio по push-уведомлению или коду из SMS.

## По СберБизнес ID

Для регистрации через сервис авторизации корпоративных клиентов СберБизнес ID у вас должна быть учетная запись [СберБизнес](https://www.sberbank.ru/ru/s_m_business) .

Чтобы зарегистрироваться в Studio по СберБизнес ID:

1. Откройте [Studio](https://developers.sber.ru/studio/) .
2. Нажмите кнопку **Войти по СберБизнес ID**.
3. В открывшемся окне введите логин и пароль от своей учетной записи СберБизнес и нажмите кнопку **Далее**.

Откроется **Личное пространство** Studio. Данные профиля заполнятся автоматически на основе данных учетной записи СберБизнес.

Для регистрации через единый сервис авторизации [Сбер ID](https://www.sberbank.ru/ru/person/dist_services/sberbankid) необязательно быть клиентом Сбербанка — достаточно иметь номер телефона российского оператора. Если такого номера у вас нет, зарегистрируйтесь [по почте](#по-почте).

## По почте

### Ограничения

С 1 декабря 2023 г. пользователи в РФ не могут регистрироваться в личном кабинете Studio с адресами иностранных почтовых сервисов (например, @gmail.com).

Это не влияет на авторизацию и использование аккаунта в Studio, если вы регистрировались с иностранной почтой до 1 декабря 2023 г.

Если вы регистрировались в Studio по Сбер ID, то продолжайте пользоваться этим способом авторизации, так как при регистрации по почте у вас создастся отдельный аккаунт. Связать аккаунты нельзя.

При регистрации по почте вы не сможете привязать устройства и [подключить ADB](https://developers.sber.ru/docs/ru/studio/space/settings/main#moi-ustroystva). Для этого используйте регистрацию по [Сбер ID](#по-сбер-id).

### Регистрация

Чтобы зарегистрироваться в Studio по почте:

1. Откройте [Studio](https://developers.sber.ru/studio/) .
2. Введите почту в поле **Почта** и нажмите **Продолжить**.
3. Придумайте и подтвердите пароль. Требования к паролю: длина от 8 символов, минимум одна цифра, минимум одна заглавная и одна строчная буква.
4. Согласитесь с условиями использования Studio и политикой конфиденциальности.
5. Нажмите **Зарегистрироваться**. При необходимости пройдите тест reCAPTCHA.

Откроется **Личное пространство** Studio. Теперь вы можете создать проект и начать работать с ним.

Для получения доступа ко всем возможностям Studio заполните профиль и подтвердите электронный адрес.

Для подтверждения электронного адреса:

1. Перейдите в [настройки профиля](https://developers.sber.ru/docs/ru/studio/profile-settings).
2. Нажмите **Подтвердить почту**, а затем **Подтвердить email**.
3. Введите код подтверждения из электронного письма. Код действует 15 минут с момента отправки письма.
4. Нажмите **Подтвердить**.

При регистрации по почте вы не сможете привязать устройства и [подключить ADB](project-creation/profile-settings.md#moi-ustroystva). Для этого используйте регистрацию по [Сбер ID](#по-сбер-id).

Если вы входили в Studio по Сбер ID, то продолжайте пользоваться этим способом авторизации, так как при регистрации по почте у вас создастся отдельный аккаунт. Пока связать аккаунты с почтой и Сбер ID нельзя.
