Каждый владелец аккаунта может добавить в свое пространство других пользователей для совместного тестирования, разработки и администрирования проектов. В личное пространство можно пригласить до 20 пользователей, в корпоративное — до 500.

## Как пригласить пользователя

Пригласить пользователей в личное или корпоративное пространство можно следующими способами:

* [Персонально](#персональное-приглашение) по номеру телефона или почте.
* По автоматически сгенерированной [ссылке](#приглашение-по-ссылке).

![Приглашение в команду](/assets/salutebot/bot-steps/project-creation/add-users/space-invite-5867141a92ea2761488d70a16a4551ef.png)

### Персональное приглашение

Персональное приглашение — это способ пригласить в свое пространство пользователей, зарегистрированных в Studio. Для этого вам понадобятся почта или номер телефона, с которыми пользователь проходил регистрацию.

Для генерации персонального приглашения:

1. Перейдите в настройки пространства на вкладку **Команда**.
2. Нажмите кнопку **Пригласить участника**.
3. Выберите тип приглашения:
   1. Телефон — российский номер телефона, привязанный к Сбер ID пользователя, если пользователь регистрировался с помощью Сбер ID.
   2. Почта — электронный адрес, если пользователь регистрировался с почтой.
4. Укажите почту или номер телефона пользователя.
5. Выберите роль для нового пользователя. Подробнее о ролях читайте в разделе [Роли пользователей](roles.md).
6. Нажмите кнопку **Сгенерировать ссылку**.
7. Отправьте полученную ссылку пользователю любым удобным способом.

Персональное приглашение действует два дня. Если не активировать приглашение, ссылка станет недействительной.

После перехода по ссылке в аккаунте пользователя отобразится новое пространство.

### Приглашение по ссылке

Приглашение по ссылке означает, что к вашему пространству сможет присоединиться любой пользователь Studio, который перейдет по сгенерированной ссылке. В целях безопасности таким пользователям по умолчанию назначается роль тестировщика с минимальными правами. Изменить роль такому пользователю можно вручную сразу после того, как он отобразится в списке вашей команды.

Для генерации ссылки:

1. Перейдите в настройки пространства на вкладку **Команда**.
2. Нажмите кнопку **Пригласить участника**.
3. На вкладке **Публичная ссылка** нажмите кнопку **Сгенерировать ссылку**.
4. Скопируйте ссылку и отправьте ее пользователю любым удобным способом.

Для одного пространства можно создать только одно приглашение по ссылке. Ссылка не имеет срока действия. Она будет активной до тех пор, пока ее не удалить. Чтобы изменить приглашение, удалите старое и сгенерируйте новое.

Если вы удалили пользователя, который пришел в пространство по ссылке, то он не сможет заново попасть в пространство по этой же ссылке. Если же пользователь вышел из пространства сам, то ссылка останется для него рабочей.
