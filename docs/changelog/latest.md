# История обновлений

### Май 2025

* Актуализировали Соглашение о работе сервиса SaluteBot в части использования GigaChat.

[Читать подробнее](https://developers.sber.ru/docs/ru/policies/salutebot-agreement)

### Март 2025

* Дополнили информацию по возможным лимитам для чат-ботов. Это поможет сделать работу сценария более корректной.

[Читать подробнее](../scenario/blocks/event.md)

* Обновили инструкцию по подключению библиотек и скриптов: дополнили описание синтаксиса.

[Читать подробнее](../scenario/scenario-management/preload-js-scripts.md)

### Январь 2025

* Теперь в аналитических отчетах можно использовать **Фильтр пустых сессий** и учитывать в своей работе только состоявшиеся диалоги с клиентами.

[Читать подробнее](../statistic-reports/reports.md)

* Обновили инструкцию по добавлению данных из чат-бота в Google-таблицу.

[Читать подробнее](../scenario/adding-data-to-google-tables.md)

### Декабрь 2024

* Добавили новые фильтры отчета **Диалоги** для чат-ботов, созданных в Code. Теперь можно оценить эффективность работы своего бота по интересующим параметрам на базе диалогов и сессий.

[Читать подробнее](../statistic-reports/reports-filter.md)

* Обновили информацию о лимитах при создании ботов на основании базы знаний (RAG).

[Читать подробнее](../bot-steps/bot-creation-by-docs.md)

* Сделали новый быстрый старт, где рассказываем о создании и настройке первого чат-бота за несколько шагов.

[Читать подробнее](../bot-steps/project-creation/overview.md)

### Октябрь 2024

* Рассказали подробнее о тарифных опциях SaluteBot.

[Читать подробнее](../tariffs/overview.md)

### Сентябрь 2024

* Рассказали о новом отчете **Воронки**, который позволяет узнать поведение клиента на каждом шаге сценария и проанализировать качество работы чат-бота.

[Читать подробнее](../statistic-reports/reports.md#воронки)

### Август 2024

* Описали новый блок **AI-ответ из базы знаний**: он позволяет пользователям получить ответ, сгенерированный нейросетью GigaChat на основе информации из загруженных документов.

[Читать подробнее](../scenario/blocks/ai-answer.md)

* Теперь при создании чат-ботов с помощью базы знаний можно загружать документы по URL.

[Читать подробнее](../bot-steps/bot-creation-by-docs.md#создание-базы-знаний)

* Добавили полезную информацию об отчетах, позволяющих анализировать работу чат-бота: **Диалоги**, **Популярные интенты** и **Активность**.

[Читать подробнее](../statistic-reports/reports.md)

### Июль 2024

* Рассказали про возможность подключать чат-бот напрямую к Telegram, без использования дополнительных чат-платформ.

[Читать подробнее](../bot-channel/telegram.md)

### Июнь 2024

* Описали способ интеграции чат-ботов с платформой EDNA.

[Читать подробнее](../edna-integration.md)

* Описали способ интеграции чат-ботов с платформой Webim.

[Читать подробнее](../webim-integration.md)

### Апрель 2024

* Теперь можно подключить сервис как для физического, так и для юридического лица с любой страницы документации. Подключите свой чат-бот сразу.

* Добавили возможность создавать SaluteBot используя имеющиеся документы – создайте ваш чат-бот не погружаясь в разработку.

[Читать подробнее](../bot-steps/bot-creation-by-docs.md)

### Январь 2024

* Создать чат-бот стало еще проще: используйте шаблон Бот-консультант GigaChat. В созданном шаблоне сценария ответы будут генерироваться автоматически.

[Читать подробнее](../bot-steps/wizard-creation.md)
