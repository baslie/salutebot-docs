# Обрабатываем число от пользователя

Обновлено 19 сентября 2024

[![](/assets/scenario/blocks/number-entry/salutebot-new.png)
SaluteBot](../../overview.md)[![](/assets/scenario/blocks/number-entry/Graph.png)
Graph](https://developers.sber.ru/docs/ru/va/graph/overview)

Блок **Ввод числа** предназначен для сохранения в переменной целого числа, которое указал пользователь. Полученную переменную [можно использовать в сценарии](https://developers.sber.ru/docs/ru/va/graph/variables/create-variables).

Если блок используется, чат-бот ожидает от пользователя число, даже если оно вводится словами. Чат-бот будет принимать такие значения, как 5, «пять» или «наверное, 5».

Тип данных в переменной — `integer`.

Блок **Ввод числа** содержит следующие поля:

* **Сообщение**. Текст обращения, которое покажет чат-бот. По умолчанию — «Введите число».
* **Имя переменной**. Имя переменной, в которую сохранится ответ пользователя. По умолчанию — `number`. Имя может содержать только цифры от 0 до 9, латинские буквы верхнего и нижнего регистров (Aa-Zz), и нижнее подчеркивание `_`. В переменных нельзя использовать ключевые слова языка JavaScript.
* **От**. Поле определяет нижний диапазон чисел, которые ожидаются от пользователя. По умолчанию `1`.
* **До**. Поле определяет верхний диапазон чисел, которые ожидаются от пользователя. По умолчанию `5`.
* **Сообщение об ошибке**. Чат-бот выводит текст сообщения, если указанное пользователем число не входит в диапазон, заданного в полях **От** и **До**. Блок может содержать несколько сообщений об ошибке. Сообщение выбирается случайным образом.

Заметили ошибку?

Выделите текст и нажмите `Ctrl` + `Enter`, чтобы сообщить нам о ней
