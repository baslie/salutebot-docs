# Сохраняем номер телефона

Обновлено 26 декабря 2023

[![](/assets/scenario/blocks/phone-number/salutebot-new.png)
SaluteBot](../../overview.md)[![](/assets/scenario/blocks/phone-number/Graph.png)
Graph](https://developers.sber.ru/docs/ru/va/graph/overview)

Блок **Номер телефона** предназначен для сохранения в переменной номера телефона, который указал пользователь. Полученную переменную можно использовать в сценарии.

Номер телефона сохраняется в переменной с российским кодом +7. Например, если пользователь указал `8 921 111-2233` чат-бот сохранит `"+79211112233"`.

Пользователь может указать номер телефона как цифрами, так и словами, устно или письменно. Код региона указывать не обязательно.

Тип данных в переменной — `string`.

Блок **Номер телефона** содержит следующие поля:

* **Сообщение**. Текст обращения, которое чат-бот покажет в чате. По умолчанию — «Введите номер телефона».
* **Имя переменной**. Имя переменной, в которую сохранится ответ пользователя. По умолчанию — `phone`. Имя может содержать только цифры от 0 до 9, латинские буквы верхнего и нижнего регистров (Aa-Zz), и нижнее подчеркивание `_`. В переменных нельзя использовать ключевые слова языка JavaScript.
* **Сообщение об ошибке**. Чат-бот выводит текст сообщения, если пользователь указал некорректный номер телефона. Блок может содержать несколько сообщений об ошибке. Сообщение выбирается случайным образом.

Заметили ошибку?

Выделите текст и нажмите `Ctrl` + `Enter`, чтобы сообщить нам о ней
