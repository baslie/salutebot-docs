# Сохраняем ответ пользователя

Обновлено 24 марта 2025

[![](/assets/scenario/blocks/text-entry/salutebot-new.png)
SaluteBot](../../overview.md)[![](/assets/scenario/blocks/text-entry/Graph.png)
Graph](https://developers.sber.ru/docs/ru/va/graph/overview)

Блок **Ввод текста** предназначен для сохранения ответа пользователя в переменной, которую можно использовать в сценарии.

Если блок добавлен, чат-бот обращается к пользователю и ждет ответа. Пользователь может как произнести ответ, так и ввести его на клавиатуре устройства. Ответ пользователя сохраняется в переменной.

Размер входящего сообщения — не более 400 символов.

Тип данных в переменной — `string`.

Блок содержит следующие поля:

* **Сообщение**. Текст обращения, которое чат-бот покажет в чате. По умолчанию — «Введите текст».
* **Имя переменной**. Имя переменной, в которую сохранится ответ пользователя. По умолчанию — `text`. Имя может содержать только цифры от 0 до 9, латинские буквы верхнего и нижнего регистров (Aa-Zz), и нижнее подчеркивание `_`. В переменных нельзя использовать ключевые слова языка JavaScript.

Наличие блока не влияет на реакцию чат-бота на ответ пользователя. Для реакции на ответ используйте блок [Интенты](intent.md).

Заметили ошибку?

Выделите текст и нажмите `Ctrl` + `Enter`, чтобы сообщить нам о ней
