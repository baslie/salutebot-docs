# Паттерны

Обновлено 15 декабря 2023

[![](/assets/bot-development/patterns/overview/salutebot-new.png)
SaluteBot](../../overview.md)[![](/assets/bot-development/patterns/overview/Code.png)
Code](https://developers.sber.ru/docs/ru/va/code/overview)

Паттерны — формальные правила, которые описывают ключевые слова и выражения, содержащиеся в запросе пользователя. С их помощью чат-бот может сопоставлять запросы пользователя с стейтами, заданными в сценарии.

Паттерны объявляются в сценарии в декларативном [теге `patterns`]/(ru/va/code/sa-dsl/tags/declarative-tags) и описываются с помощью [базовых](https://developers.sber.ru/docs/ru/va/code/nlp/patterns/base-patterns) и [расширенных](https://developers.sber.ru/docs/ru/va/code/nlp/patterns/advanced-patterns) элементов.
