# Справочники меток

Обновлено 6 марта 2024

[![](/assets/bot-development/project-structure/tag-list/salutebot-new.png)
SaluteBot](../../overview.md)[![](/assets/bot-development/project-structure/tag-list/Code.png)
Code](https://developers.sber.ru/docs/ru/va/code/overview)

Для перехода к справочнику меток нажмите **Настройки** в левом верхнем углу главной страницы → **Настройки сервисов** → **SmartApp Graph / Code** → вкладка **Справочник меток**.

Здесь вы можете создать группы меток для сообщений или отредактировать группы **Метки для сессий**, **Метки для задач**, **Задачи — приоритеты**, **Задачи — состояния**.

Проставляйте метки для классификации сессий, задач или сообщений.

Для создания группы меток для сообщений:

1. Нажмите **Добавить группу меток**.
2. Укажите номер группы и название, переведите переключатель в активное положение.
3. Нажмите **Добавить** под названием группы, укажите номер и название метки. Переведите переключатель в активное положение. Теперь вы можете проставлять метки для сообщений.

Заметили ошибку?

Выделите текст и нажмите `Ctrl` + `Enter`, чтобы сообщить нам о ней
