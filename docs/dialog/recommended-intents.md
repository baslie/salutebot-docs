# Проверяем обращения пользователей

Обновлено 19 сентября 2024

[![](/assets/dialog/recommended-intents/salutebot-new.png)
SaluteBot](../overview.md)

При работе чат-бота собирается статистика диалогов с пользователями. На основе собранной статистики предлагается список готовых интентов и список реплик для создания собственных интентов.

Рекомендованные интенты доступны только для проектов SaluteBot.

Подробнее о работе с [рекомендованными репликами](adding-recommend-intent.md).
