# Используем автоматическую генерацию реплик

Обновлено 19 сентября 2024

[![](/assets/dialog/intents-autogeneration/salutebot-new.png)
SaluteBot](../overview.md)[![](/assets/dialog/intents-autogeneration/Code.png)
Code](https://developers.sber.ru/docs/ru/va/code/overview)[![](/assets/dialog/intents-autogeneration/Graph.png)
Graph](https://developers.sber.ru/docs/ru/va/graph/overview)

Для упрощения создания интентов при разработке сценария, возможно использовать автоматическую генерацию реплик в интентах.

В этом случае значительно упрощается создание собственных интентов: вам не нужно искать и придумывать примеры реплик для интентов, достаточно просто дать задание на генерацию и получить список готовых примеров для обучения и использования.

Количество попыток автогенерации ограничено условиями тарифа.
На бесплатном тарифе доступно не более 5 попыток для всех проектов.

Code

Чтобы сгенерировать примеры для интента:

1. Откройте сценарий бота в Graph.
2. Добавьте новый блок **Интенты** или откройте параметры нужного блока в сценарии.
3. Выберите пункт **Создать интент** с плашкой **Нейросеть**.
4. В открывшемся окне укажите **Имя** (тематику) разговора и приведите несколько примеров в поле **Реплики**. Реплики - это фразы пользователя, в ответ на которые выполнится шаг сценария, связанный с интентом.

Например, тематикой может быть "Покупка подержанных автомобилей", а репликой - "Купить б/у машину в Москве".

5. Нажмите **Придумать реплики**.

Для добавления реплик вручную достаточно нажать Enter, в этом случае автогенерация использоваться не будет.

6. Дождитесь окончания процесса, в результате которого вам будут представлены сгенерированные с помощью нейросети реплики, подходящие под тематику вашего интента. Вы можете удалить неподходящие вам реплики и повторить генерацию. При повторной генерации добавятся дополнительные примеры.
7. Нажмите **Создать и добавить в блок**.
8. При необходимости добавьте еще один интент или отредактируйте уже созданный. Определите, будет ли сгенерированный интент применен для всего сценария или конкретного шага.
9. Сохраните изменения, нажмите **Создать блок**.

Чтобы добавить автосгенерированные интенты:

1. Откройте сценарий бота в Code.
2. Перейдите в раздел **Редактор** → **Интенты**.
3. Добавьте новый интент или откройте параметры существующего.
4. Укажите **Название** и **Тематику интента**.
5. Добавьте **Тренировочные фразы** как пример реплик для автогенерации.
6. Нажмите **Придумать реплики**. Дождитесь окончания процесса, в результате которого вам будут представлены сгенерированные с помощью нейросети реплики, подходящие под тематику вашего интента.
7. Сохраните изменения, нажмите **Тестировать**.
